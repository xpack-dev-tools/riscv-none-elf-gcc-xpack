The initial name of the package was
[GNU MCU Eclipse RISC-V Embedded GCC](https://github.com/ilg-archived/riscv-none-gcc)
and was first published on `npmjs.com` as
[`@gnu-mcu-eclipse/riscv-none-gcc`](https://www.npmjs.com/package/@gnu-mcu-eclipse/riscv-none-gcc).

The early releases followed the SiFive releases and used the SiFive forks
of GCC/GDB/etc.

### July 2017

The first **GNU MCU Eclipse RISC-V Embedded GCC** release was
[v7.1.1-1-20170702](/blog/2017/07/02/riscv-none-gcc-v7-1-1-1-20170702-released/).

### July 2019

<HistoryRenameXpack/> and the npm scope was changed from `@gnu-mcu-eclipse` to
[`@xpack-dev-tools`](https://www.npmjs.com/package/@xpack-dev-tools/riscv-none-elf-gcc)
(in [v8.2.0-3.1](/blog/2019/07/31/riscv-none-embed-gcc-v8-2-0-3-1-released/)).

### July 2020

<HistoryArmLinuxAdded/> (in [v8.3.0-1.2](/blog/2020/07/04/riscv-none-embed-gcc-v8-3-0-1-2-released/)).

### October 2020

After extensive experimentation, support for Python
scripting was added to GDB, using an embedded Python 3 instance
(in [v8.3.0-2.2](/blog/2020/10/13/riscv-none-embed-gcc-v8-3-0-2-2-released/)).

### October 2021

<HistoryWindowsUcrt/> (in [v10.1.0-1.2](/blog/2021/11/04/riscv-none-embed-gcc-v10-1-0-1-2-released/)).

### May 2022

The projects switched from the SiFive forks to the official GNU sources (in [v11.3.0-1](/blog/2022/05/14/riscv-none-elf-gcc-v11-3-0-1-released/)).

<HistoryAppleSiliconAdded/> (in [v11.3.0-1](/blog/2022/05/14/riscv-none-elf-gcc-v11-3-0-1-released/)).

<History32bitDiscontinued platforms="x86 Windows and x86 GNU/Linux"/> (in [v11.3.0-1](/blog/2022/05/14/riscv-none-elf-gcc-v11-3-0-1-released/)).
